## [Architectural_Laws]
- **Export_Law**: One module per feature folder. Controllers handle HTTP. Services handle business logic. Export services from modules for cross-module DI. Use barrel exports.
- **Transformation_Law**: Dependency Injection via constructor injection (TypeScript `experimentalDecorators`). Request lifecycle: Middleware → Guards → Interceptors (before) → Pipes → Handler → Interceptors (after) → Exception Filters.
- **Propagation_Law**: Throw `HttpException` subclasses for errors. Exception filters catch and transform errors. Use `@UseFilters()`, `@UseGuards()`, `@UseInterceptors()`, `@UsePipes()` at controller/method level. Global registration via `app.useGlobal*()`.

async function bootstrap() {
  const app = await NestFactory.create(AppModule)
  app.enableCors()
  app.setGlobalPrefix("api")
  app.useGlobalPipes(new ValidationPipe({ whitelist: true, transform: true }))
  app.useGlobalFilters(new HttpExceptionFilter())
  app.useGlobalInterceptors(new LoggingInterceptor())
